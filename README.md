# Web Technologies and Platforms  
## Lab Assignment – 02  
### HTML and CSS Fundamentals Using Selectors

---

## Assignment Overview
This lab assignment focuses on building a **Student Profile Web Page** using **semantic HTML** and **external CSS**.  
The goal of this assignment is to understand and apply different **CSS selectors** while maintaining proper separation between content and presentation.

Advanced CSS features and JavaScript are strictly avoided as per the assignment requirements.

---

## Learning Objectives
By completing this lab, the student demonstrates the ability to:
- Create a well-structured HTML document
- Use semantic HTML elements correctly
- Apply CSS using:
  - Element selectors
  - Class selectors
  - ID selectors
  - Group selectors
  - Descendant selectors
- Maintain separation of content and styling
- Inspect and analyze CSS using browser developer tools

---

## Tools and Technologies Used
